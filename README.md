# nodejs-custom-http-api-topic3

## Project Overview
This repository contains the Node JS Hands-on Topic 3 assessment project based on building a custom HTTP API using Node.js.

## Features Implemented
- Custom HTTP server
- API routing
- JSON responses
- Request handling
- Localhost testing

## Steps Performed
1. Installed Node.js and npm
2. Created a Node.js project
3. Initialized the project using npm
4. Used the built-in http module
5. Created API routes
6. Started server using:
   node index.js
7. Tested responses in browser and terminal

## Technologies Used
- Node.js
- JavaScript
- HTTP Module

## Challenges Faced
- Port conflicts
- Incorrect route handling
- Server restart issues

## Resolution
Resolved issues by correcting routes, checking port configuration, and restarting the server properly.

## Output
The custom HTTP API server executed successfully and responses were verified locally.
